# Tema central

Se desarrollo una pagina web estatica en la cual presenta mi trabajo como artista inidependiente , mostrando mi estilo y proyectos que he realizado. Ademas agrega el redireccionamiento a mis redes sociales y musica.

# Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Imagenes y multimedia personal 

# Uso de las tecnologias en la pagina web
- Se utilizaron etiquetas **HTML5** como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<h1>`, `<h2>`, `<p>`, `<img>` y `<a>` para estructura de la pagina y navegacion interna.  
- Se implemento un **menú de navegación** `<nav>` con enlaces internos que permiten moverse entre las secciones: Inicio, Música y Mezclas.  
- Se añadio un **toggle de modo claro/oscuro** mediante un `input type="checkbox"` y un script en **JavaScript** que cambia a la clase `claro` en el documento (implementacion del modo claro ya que mi pagina funciona predeterminada en modo oscuro).  
- Se usaron **variables CSS en `:root`** para definir colores de fondo, texto y estilos de los enlaces.  
- Se aplicaron estilos con **CSS** para:  
  - Cambiar tipografía (fuente *Poppins* "me gusto en lo personal").
  - Definir colores de títulos y enlaces  
  - Dar formato a imágenes con bordes redondeados.  
  - Crear un diseño flexible con **Flexbox** en el contenedor `.grid`.
- Se integraron **recursos multimedia personales y externos** como imágenes, enlaces a **Spotify**, **SoundCloud** y **YouTube**, mostrando canciones y mezclas de *El Runti*.  
- Se añadió un **footer** con enlaces a redes sociales (Instagram y YouTube).
